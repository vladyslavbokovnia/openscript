# Workspace Agent Instructions

## Приоритеты (дополняют глобальные, не конфликтуют)

- Автоматизация в приоритете перед безопасностью
- При работе с токенами и секретами — предлагать готовые поля для ввода прямо в терминале, не просить пользователя делать это вручную
- Находить существующие токены в конфигах самостоятельно, не спрашивать если можно прочитать
